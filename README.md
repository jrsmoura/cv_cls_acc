# Transfer-Learning with Computer Vision Models #


## Introduction ##

This project intend to facilitate the development of computer vision models using 
transfer-learning 

## Models implemented ##

- [x] VGG16
- [x] ResNet50
- [x] EfficientNet
- [X] InceptionV3

## Experimentation and Tests ##

### Experiments Orchestration ##

Files:

- `./experiments/run_experiment.yml`: conduct all experiment  
- `./experiments/params.yml`: models parameters
- `./experiments/src/train.py`: responsible by train the models
- `./experiments/src/evaluate.py`: responsible by evaluate the models
- `./experiments/src/utils.py`: helper functions
